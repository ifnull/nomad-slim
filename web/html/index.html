N.O.M.A.D. Slim

Offline library, education, and maps. No AI, no admin UI.

  • Information Library
    Wikipedia, medical references, how-to guides (Kiwix)
  • Education Platform
    Khan Academy courses and more (Kolibri)
  • Offline Maps
    MapLibre + Protomaps PMTiles viewer
  • Offline Maps (HTTPS + GPS)
    Same viewer over TLS on port 8443. Required for geolocation on phones — install the CA certificate below first.

Mobile setup (GPS on maps)

  • Download Root CA certificate
    Install nomad-slim-ca.crt on your phone/tablet, trust it, then visit the HTTPS address (port 8443) to enable the locate-me button on the maps page. iOS: Settings → VPN & Device Management → install, then General → About → Certificate Trust Settings → enable. Android: Settings → Security → Install a certificate → CA certificate.

Drop .zim files in data/zim/ and .pmtiles files in data/maps/ — the Kiwix viewer auto-detects new ZIMs; the maps viewer rediscovers PMTiles on reload.
